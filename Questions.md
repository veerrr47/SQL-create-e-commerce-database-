Tasks & Questions

Q1 (5 Marks):
Write a stored procedure GetPatientAppointments that takes a PatientID as input and returns all appointments for that patient, including Appointment Date, Doctor Name, Specialization, Diagnosis, and Fees.

Q2 (5 Marks):
Write a stored procedure GetDoctorAppointments that takes a DoctorID as input and returns all appointments for that doctor, including Patient Name, Appointment Date, and Diagnosis.

Q3 (5 Marks):
Write a stored procedure GetAppointmentsInRange that takes two dates as input and returns all appointments within that range, including Patient Name, Doctor Name, and Diagnosis.

Q4 (5 Marks):
Write a stored procedure GetTotalEarningsByDoctor that takes a DoctorID as input and returns the total earnings from all appointments for that doctor.

Q5 (5 Marks):
Write a stored procedure GetMostVisitedDoctor that returns the doctor with the highest number of appointments, along with the total number of patients seen.
